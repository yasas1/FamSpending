About

FmS

FmS is a Family Spending tracking application that is focused only on daily spending along with customizable Categories and Members. This includes Calendar Spending Home view, New Spending, Category-Member, Summary view, and spending day view.

Calendar Spending Home View

You can clearly see here all the entered spending within the past 12 months on a calendar view. These spendings are emphasized with blue color on entered dates. Total spendings for each day and for the selected month are displayed. Also, you can see a listed view and a summary of spendings for a selected day using the calendar.

Category-Member

Category items help you to mention that this was spent what for. Examples:

Examples
- Food
- Medicine
- Bills

Member items help to mention for whom you have done the spending Examples:

Examples
- Family
- Me
- Outsource

For each "categories" and "members" will have a maximum of 10

New Spending

Use this to add new spending for a selected date by filling out the following details.

  1. Date : date of the spending
  2. Member : for whom you spend
  3. Category : type of the spending
  4. Note : breif description
  5. Essential : imprtant or not
  6. Amount : Spending amount
Summary

Total spending will display for the selected summary type with the category, member, and essentiality. By default, the current date summary will be shown ( if any spending has done).

Summary types
- Today, Yesterday
- This week, Last week
- This month, Last month
- This year, Last year
Day View

Here, you can see spendings for the selected date. Also, you can manage spendings by editing and deleting.


Next updates will be included with date range spending, summary charts, selected date range summary, etc.
