About

FmS

FmS is a Family Spending tracking application for a family which is focused only on daily spending with managing customizable Categories and Members. This includes Calendar Spending Home view, New Spending, Category-Member, Summary view and spending day view

Calendar Spending Home View

You can clearly see here all the entered spending within past 12 month on a calendar view. These spendings are emphasized with blue color on entered dates. Total spendings for each day and for the selected month are displayed. Also you can view listed and summary spending for a selected day from this.

Category-Member

Categories help you to mention that this was spent what for. Examples:

Examples
- Food
- Medicine
- Bills

Members help you to mention that this was spent for a family member. Examples:

Examples
- Family
- Me
- Outsource

Both categories and members can have 10 maximum limit for each

New Spending

In here you can input a new spening for a selected date.

  1. Date : Relevant date
  2. Member : This is for who
  3. Category : This is for what
  4. Note : Breif description
  5. Essential : This is necessary or not
  6. Amount : Spending amount
Summary

Total spendings will be displayed by grouping with category, member and essential for seletected type. Default there will be shown summary for today if there any.

Summary types
- Today, Yesterday
- This week, Last week
- This month, Last month
- This year, Last year
Day View

In here, you can see spendings for selected date. Also you can manage spendings by editing and deleting.


Next updates will be included with date range spending, summary charts, selected date range summary, etc.
